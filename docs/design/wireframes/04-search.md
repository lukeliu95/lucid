## 关键决策

- **三段结果垂直堆**(视频 > 人物 > 主题)· 不用 tab(用户一次看全)
- **关键词 / 语义 toggle 显式切**(MVP 不自动判定 · 避免黑盒)
- **`<mark>` 暖橘高亮** 命中词
- 输入框 300ms debounce
- 0 命中 → "没找到 <q>...</q> 相关内容 · 试试 [Agent] [创业] [芯片]" 推荐胶囊

## 数据态

- loading: 三段 skeleton
- empty: 提示 + 主题胶囊推荐
- error: "搜索服务暂时不可用 · 重试"

## a11y

- 搜索框 `aria-label="搜索视频、人物、主题"`
- 结果分段 `<h2>` · `aria-live="polite"` 通报命中数
- 关键词高亮 `<mark>` semantic
